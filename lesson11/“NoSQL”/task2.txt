При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, 
поиск электронного адреса пользователя по его имени.


127.0.0.1:6379> set email qwerti@yandex.ru
OK
127.0.0.1:6379> get email
"qwerti@yandex.ru"
127.0.0.1:6379> set Sergey serega@mail.ru
OK
127.0.0.1:6379> get Sergey
"serega@mail.ru"
127.0.0.1:6379>
